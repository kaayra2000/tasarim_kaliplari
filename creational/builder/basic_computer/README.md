# basic_computer

Director + farklı builder'larla iki bilgisayar konfigürasyonu oluşturan çalışır örnek.

# Derleyip Çalıştırma

Bu klasördeyken:

```bash
g++ -std=c++17 code.cpp -o app
./app
```
